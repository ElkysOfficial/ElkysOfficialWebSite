# Team Photos

Esta pasta contém as fotos dos membros da equipe.

## Requisitos das Imagens

Para melhor aparência e consistência visual:

- **Formato:** JPG ou PNG
- **Dimensões recomendadas:** 800x800px (quadrado)
- **Tamanho máximo:** 500KB por imagem
- **Fundo:** Preferencialmente neutro ou desfocado
- **Qualidade:** Alta resolução

## Nomenclatura dos Arquivos

As fotos devem ser nomeadas como:
- `member-1.jpg`
- `member-2.jpg`
- `member-3.jpg`
- `member-4.jpg`
- `member-5.jpg`
- `member-6.jpg`
- `member-7.jpg`
- `member-8.jpg`

## Atualizar Informações da Equipe

Para atualizar as informações dos membros, edite o arquivo:
`src/components/Team.tsx`

Altere o array `teamMembers` com as informações corretas:
- `name`: Nome completo
- `role`: Cargo/Posição
- `bio`: Breve descrição (1-2 frases)
- `image`: Caminho da foto (ex: `/imgs/team/member-1.jpg`)
- `linkedin`: URL do perfil no LinkedIn
